 DentalApp - Gestión de Citas 🦷

Sistema sencillo de gestión de turnos dentales compuesto por una app **Android** y un backend en **Java**.

## 📋 Funcionalidades
- **Registro de citas:** Formulario para elegir fecha y hora.
- **Consulta:** Listado de citas agendadas desde el servidor.
- **Cancelación:** Eliminar turnos existentes.

## 🛠️ Tecnologías
- **App:** Android Studio + HTTPok para las peticiones.
- **Servidor:** Java Servlets ejecutándose en **Apache Tomcat**.
- **Datos:** Intercambio de información en formato **JSON**.

## 🚀 Instalación rápida
1. Sube el proyecto web a tu Tomcat.
2. Ajusta la IP del servidor en el código de Android.
3. Ejecuta la app y gestiona tus citas.
